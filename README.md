# Solar Panel Power Meter

## Project Purpose

This project measures the output of a small solar panel, calculates power, and displays the results in real time.

## System Overview

The system measures:

- Solar panel voltage
- Load current
- Output power

The design uses:

- Voltage divider for voltage sensing
- Shunt resistor for current sensing
- Op-amp circuit for signal conditioning
- Arduino-compatible controller for measurement and calculation
- LCD display for output
- Solar panel as the input source

## Planned Features

- Measure voltage from a small solar panel
- Measure current through a load
- Calculate power using measured voltage and current
- Display voltage, current, and power on an LCD
- Compare performance under different lighting and load conditions
- Document circuit behavior, test data, and design changes

## Technical Focus

- Circuit analysis
- Voltage dividers
- Current sensing
- Shunt resistor measurements
- Op-amp signal conditioning
- Embedded control
- Arduino programming
- LCD display integration
